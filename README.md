# RiverD SoundIO
Dynamic and static D bindings for SoundIO library

## Usage
```d
import riverd.soundio;

void main()
{
	// Load the SoundIO library.
	dylib_load_soundio();

	// Now SoundIO functions can be called.
	...
}
```
